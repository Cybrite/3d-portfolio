# 3D Portfolio

A modern and interactive 3D portfolio website built with React and Vite.

## Technologies Used

- React
- Vite
- Three.js (if you're using 3D features)
- Miximo

## Features

- Interactive 3D elements
- Responsive design
- Fast page loading with Vite's HMR (Hot Module Replacement)
